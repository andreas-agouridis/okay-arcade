## Before Asking for Help

### Try These Steps First

1. **Clear browser cache**
   - Chrome: Ctrl+Shift+Delete
   - Firefox: Ctrl+Shift+Delete
   - Safari: Cmd+Option+E

2. **Check browser console**
   - Press F12
   - Click "Console" tab
   - Look for red error messages

3. **Try a different browser**
   - Chrome, Firefox, Safari, Edge

4. **Check documentation**
   - Search the docs for your issue

5. **Try a local server**
   ```bash
   python -m http.server 8000
   ```

6. **Check the FAQ**
   - Your question may already be answered

## Common Issues Quick Fixes

| Issue | Quick Fix |
|-------|-----------|
| Games not loading | Use HTTPS URLs |
| Blank page | Check console errors |
| Favorites not saving | Enable localStorage |
| Modal not opening | Check game data format |
| Icons missing | Check internet/Fort Awesome CDN |
| Theme not toggling | Check JavaScript console |
