> Type commands to play games. No graphics, no fluff. > Available commands:
  list / ls - show all available games
  play [number/name] - play a game by ID or name
  search [term] - search games by keyword
  fav / favorites - show your favorite games
  addfav [id/name] - add game to favorites
  rmfav [id/name] - remove from favorites
  random / r - play random game
  filter [category] - filter by: action, racing, puzzle, sports
  clear / cls - clear screen
  stats / status - show statistics
  help / ? - show this help
  exit / quit - (just clears, keep playing)
